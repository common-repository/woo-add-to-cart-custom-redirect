Redirect customers to custom URL after a WooCommerce product is added to the cart.

== Description ==

Redirect customers to custom URL after a WooCommerce product is added to the cart.

== Installation ==


1. Upload the plugin files to the '/wp-content/plugins/woo-add-to-cart-custom-redirect' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress



== Frequently Asked Questions ==

= Is WooCommerce is required? =

Yes, this is WooCommerce addon, so WooCommerce is required.
